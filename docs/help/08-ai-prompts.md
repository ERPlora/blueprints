# Asistente IA — Guía de Prompts

El asistente de IA puede configurar tu negocio, crear productos, gestionar empleados y mucho más. Aquí tienes ejemplos de prompts que puedes copiar y adaptar.

## Configuración Inicial

Estos prompts configuran tu negocio desde cero:

| Prompt | Qué hace |
|--------|----------|
| `Tengo un restaurante en Madrid` | Configura región, instala módulos, crea productos, impuestos y compliance |
| `Configura una cafetería en Barcelona` | Setup completo para cafetería |
| `Tengo una peluquería en Sevilla` | Setup completo para salón de belleza |
| `Set up a bar in Valencia` | Funciona también en inglés |
| `Tengo una pizzería en Lisboa, Portugal` | Configura con IVA portugués |

> Después de la configuración inicial, el asistente preguntará el nombre del negocio y la dirección.

## Productos e Inventario

### Instalar catálogo predefinido

| Prompt | Qué hace |
|--------|----------|
| `Instala el catálogo de productos de restaurante` | Importa ~50 productos con imágenes y precios |
| `Instala solo las tapas del catálogo` | Importa solo los productos de la categoría tapas |
| `¿Qué productos tiene el catálogo de pizzería?` | Muestra los productos disponibles sin instalar |
| `¿Qué catálogos hay disponibles?` | Lista los tipos de negocio con catálogo |

### Crear productos manualmente

| Prompt | Qué hace |
|--------|----------|
| `Crea un producto: Tarta de Santiago 6.50€ en Postres` | Crea un producto con precio y categoría |
| `Añade estos productos: Café 1.50€, Tostada 2€, Zumo 3€` | Crea varios productos a la vez |
| `Crea la categoría Desayunos con el icono sunny-outline` | Crea una nueva categoría |

### Gestionar stock

| Prompt | Qué hace |
|--------|----------|
| `Muéstrame los productos con stock bajo` | Lista productos por debajo del umbral |
| `Actualiza el stock de Café solo a 100 unidades` | Ajusta stock de un producto |
| `Exporta los productos a CSV` | Genera un archivo CSV descargable |

## Mesas y Zonas

| Prompt | Qué hace |
|--------|----------|
| `Crea las zonas: Terraza (8 mesas), Interior (10 mesas), Barra (5 mesas)` | Crea zonas con mesas de una vez |
| `Añade 4 mesas más a la terraza` | Añade mesas a una zona existente |
| `Muéstrame las zonas y mesas configuradas` | Lista la configuración actual |

## Empleados y Roles

|--------|----------|

## Configuración del Sistema

| Prompt | Qué hace |
|--------|----------|
| `¿Qué módulos tengo instalados?` | Lista módulos activos |
| `Muéstrame la configuración actual del negocio` | Resumen de configuración |
| `¿Qué módulos están disponibles?` | Lista módulos del marketplace |

## Consejos

- **Sé específico** — "Crea un café solo a 1.50€ en la categoría Cafés" funciona mejor que "añade un café"
- **Usa listas** — Puedes crear varios elementos en un solo mensaje
- **Pide confirmación** — El asistente siempre pide confirmación antes de hacer cambios
- **Edita antes de enviar** — Copia estos prompts y adáptalos a tu negocio
- **Mensajes restantes** — Consulta el contador en la barra del chat para ver cuántos mensajes te quedan
